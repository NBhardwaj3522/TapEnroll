# Build MVP for Grocery Pilot (TapEnroll)

This project is an MVP for a grocery NFC enrollment pilot ("TapEnroll").

## Overview
The goal of this app is to support an NFC-driven grocery pilot that enables:
- Enrollment flows
- Barcode / QR generation
- Messaging and verification via Twilio
- Data persistence via Supabase

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Supabase
- Twilio
